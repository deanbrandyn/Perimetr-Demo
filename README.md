# PERIMETR – iOS Demo (3D “Eagle View”) – GitHub Pages

This build uses **Mapbox GL JS** for a true **3D tilt / eagle view** and better address accuracy.

## You need a Mapbox token (public)
1. Create a free Mapbox account.
2. Create a **public access token** (starts with `pk.`).
3. Paste it on the login screen and tap **SAVE TOKEN**.

## Deploy to GitHub Pages
1. Create a new GitHub repo.
2. Upload all files in this ZIP to the repo root.
3. GitHub → Settings → Pages → Deploy from branch → `main` / `(root)`

## iPhone
Safari → Share → **Add to Home Screen** (optional)

## Notes
- This is still a **demo** (no backend).
- Mapbox usage is subject to your Mapbox plan/limits.
